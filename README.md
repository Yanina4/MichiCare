# 🐾 Veterinaria - Sitio Web

Bienvenido/a al repositorio de la página web de una veterinaria.  
Este proyecto fue creado con el objetivo de brindar información sobre servicios veterinarios, cuidado de mascotas y gestión de turnos.

---

## 🌐 Vista del proyecto

👉 Este proyecto puede ser visualizado como página web usando GitHub Pages (si está activado).

---

## 🛠️ Tecnologías utilizadas

- HTML5  
- CSS3  
- JavaScript  

---

## 📌 Funcionalidades

- Página principal con información de la veterinaria  
- Sección de servicios (consultas, vacunación, etc.)  
- Información sobre cuidado de mascotas 🐶🐱  
- Formulario de contacto o turnos (si lo agregaste)  
- Diseño responsivo adaptable a celular y PC
